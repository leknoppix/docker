# Création des dossiers vboxsf
Dans virtualbox, partager les dossiers suivants:

  - Dossier contenant les sites web (php, html, css...) `www` 
  - Dossier contenant les sources ruby `ruby`
  - Dossier contenant en local les sources de la base de donnée `mysql`
  - Dossier contenant les sources nodejs  `nodejs`
  - Dossier contenant les configurations (apache, nginx....) `config`
  - Dossier contenant les logs nginx `logs`
